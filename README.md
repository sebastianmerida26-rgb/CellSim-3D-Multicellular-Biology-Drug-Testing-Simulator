## 🧪 How the Model Works

This simulator uses several parts to shape the result:

- Cell cycle rules based on Novak-Tyson style ODE logic
- Diffusion behavior based on Fick’s law
- Drug response logic based on pharmacodynamics
- PK/PD style dose response behavior
- Agent-based cell movement and interaction

This means the app can show both cell-level behavior and tissue-level drug spread in one model.
